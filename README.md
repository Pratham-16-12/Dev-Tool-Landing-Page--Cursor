# Cursor Landing Page Clone

A desktop-first landing page recreation of [cursor.com](https://cursor.com) for Web Dev Cohort 2026.

## About This Project

Static HTML/CSS recreation of the Cursor AI Code Editor landing page with visual and structural accuracy.

## Sections Recreated

| #   | Section           | Description                                          |
| --- | ----------------- | ---------------------------------------------------- |
| 1   | Navigation Bar    | Fixed dark navbar with logo, nav links, Download CTA |
| 2   | Hero Section      | Serif italic headline + product screenshot           |
| 3   | Trusted By        | 8 company logos in card grid                         |
| 4   | Feature Sections  | 3 blocks with alternating text/image layout          |
| 5   | Testimonials      | 6 quote cards with author images                     |
| 6   | Stay on Frontier  | 3 feature cards with images                          |
| 7   | Changelog         | List of updates with dates                           |
| 8   | Team/About        | Team photo + description                             |
| 9   | Recent Highlights | Blog/news section                                    |
| 10  | Footer            | Multi-column links + SOC 2 badge                     |

## Constraints Followed

- HTML and CSS only
- No JavaScript
- No TailwindCSS
- No animations
- Desktop-only (no responsiveness)
